Frequent Questions

You can register by going to the Course section of the site. Find the Course you like and click "Enroll Now" to begin Enrollment.
Go to the 'My Courses' tab on Student Page to access all courses you are enrolled in. Click on the Classroom link to access the Classroom page
Classroom page offers a Discussions page to allow students enrolled in the class to engage in discussions with the Tutor as well as other Stuidents in the class.
The Classroom page has a link to the Video page to join a Video call for virtual online lessons.
While enrolling you will be prompted to make a payment after which you will see the course in the 'My Courses' section of the student Page.
The classroom schedule page shows the schedule of the classes for the course.
You must first register as a Student to be able to register for classes. Use the 'Become a Student' Link to register as a student.
You must first register as a Tutor to be able to create new Courses. Use the 'Become a Tutor' Link to register as a Tutor.
